# Animal Fact App (React Native)

This is a React Native app built using Expo.

The app allows users to:
- Enter an animal (currently supports "cat")
- Enter the number of facts to retrieve
- Fetch real data from an API and display it on the screen

Features:
- Controlled TextInput fields using useState
- Input validation with error messages
- API call using fetch() and async/await
- Loading state while fetching data
- Error handling for invalid input or API issues

This project demonstrates key React Native concepts such as state management, controlled components, validation, and API integration.
